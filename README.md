# Sklo

**Chatbots that work while you don't.**

Sklo allows businesses to build specialized AI chatbots that answer customer questions, qualify leads, and handle FAQs automatically - no coding required.

## How to use Sklo

1. **Sign Up/Login**: Create an account to access your dashboard.
2. **Create a Chatbot**: Define your chatbot's name, role, and personality.
3. **Add Knowledge**: Upload documents/images or add facts to your chatbot's knowledge base so it knows answers to common questions.
4. **Set Constraints**: Define rules (e.g., "No politics", "Only mention features X, Y, Z") to keep the bot on track.
5. **Monitor & Optimize**: Watch conversations in real-time with our Observer Mode.
